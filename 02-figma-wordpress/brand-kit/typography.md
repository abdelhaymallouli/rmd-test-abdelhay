# Atlas Coffee Tangier — Typography

The brand uses **2 typefaces only**, both free from Google Fonts.

| Role      | Family       | Source                                       | Weights to use              |
| --------- | ------------ | -------------------------------------------- | --------------------------- |
| Headings  | **Fraunces** | <https://fonts.google.com/specimen/Fraunces> | 600 (semi-bold), 700 (bold) |
| Body & UI | **Inter**    | <https://fonts.google.com/specimen/Inter>    | 400 (regular), 500 (medium) |

## Why these two

* **Fraunces** is a contemporary serif with personality — warm, slightly quirky, food / artisan-friendly. It evokes craft without being old-fashioned.
* **Inter** is a modern, neutral sans-serif designed for screen UI. It pairs cleanly with Fraunces and does not steal the show.

## Type scale (suggested, you can adjust)

| Element                 | Family   | Size (desktop) | Size (mobile) | Weight | Line-height |
| ----------------------- | -------- | -------------- | ------------- | ------ | ----------- |
| H1 (hero headline)      | Fraunces | 56 px          | 36 px         | 700    | 1.1         |
| H2 (section headline)   | Fraunces | 36 px          | 28 px         | 600    | 1.2         |
| H3 (feature card title) | Fraunces | 22 px          | 20 px         | 600    | 1.3         |
| Body (paragraph)        | Inter    | 17 px          | 16 px         | 400    | 1.6         |
| Small / caption         | Inter    | 14 px          | 14 px         | 500    | 1.5         |
| CTA button label        | Inter    | 16 px          | 16 px         | 500    | 1           |

## Rules

* Do **not** mix in a third font (no Roboto, no Poppins, no Montserrat as a "fix").
* Do **not** use Fraunces for body copy. It is a heading face.
* Do **not** use ALL CAPS on headings except optionally for the CTA button label.
